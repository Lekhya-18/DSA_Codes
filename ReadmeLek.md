# DSA_Codes
This repository contains my personal implementations of Data Structures & Algorithms in C/C++. It covers:
- Searching (Linear, Binary)
- Sorting (Bubble, Quick, Merge, Heap…)
- Hashing techniques
- Linear Data Structures (Stack, Queue, Linked List…)
- Non-Linear Data Structures (Binary Tree, BST, Graphs…)

## Getting Started
1. Clone the repository
2. Build the code: e.g. `gcc path/to/file.c -o output`
3. Run: `./output`

## Folder Structure
- `Algo’s/` : Basic algorithms
- `Linear DS/` : Codes related to <STACKS>,<QUEUES>,<LINKED LIST>…
- `Non-Linear DS/ Trees/` : Codes related <TREES>,<HEAPS>,<GRAPHS> …
